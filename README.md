# Elektronické dálniční známky
### This SWI2 schools project aims to provide a frontend to basic electronic vignettes management. 



### Dev environment 
* `Node >= 10.16`
* `npm >= 5.6 `

To verify: 
* `npm -v`
* `node -v`

### Available Scripts

Run project:
#### `npm start`

Update project dependencies

#### `npm update`

Test
#### `npx cypress open`

Production bundled build
#### `npm build`
